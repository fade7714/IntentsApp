Zadania do wykonania:

1. Twoim zadaniem jest wykonać obsługę przycisków widocznych w głownej
aktywności. To znaczy każdy przycisk ma mieć zaimplementowaną metodę:
setOnClickListener() oraz onClick().

2. Poprawnie zaimplementować pierwszą aktywność.

3. Poprawnie zaimplementować działanie drugiej aktywności uruchamiającej intencje
aplikacje przeglądarki. Nie zapomnij pobrać tekstu z obiektu TextView widocznego
w głównej aktywności.

4. Poprawnie zaimplementować działanie aktywności odpowiedzialnej za pobieranie
lokalizacji. Pamiętaj tutaj o opcji blokowania naprzemiennego przycisków start i stop
odpowiednimi właściwościami podanymi powyżej w tekście.
